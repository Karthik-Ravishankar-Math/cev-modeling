# cev-modeling
Monte Carlo modeling of CEV model for stock paths.
There are 5 python notebooks:
1) Stock Paths: This notebook generates several stock paths for the CEV model, and visualizes the stock paths for different beta values as a graph.
2) Option Pricing: The price of a call/put option for the CEV model is derived using Monte Carlo methods as there are no general closed form solutions
3) Delta: We find Delta for the CEV model using a numerical/Monte Carlo approach. Unlike in GBM there is no nice closed form solution.
4) Implied Volatility: The implied volatility as a function of strike price is modelled for different values of beta. According to theory, a negative beta should result in an implied volatility smile, but we were unable to tune the parameters to get this to work.
5) Gold Data: The stock paths for gold over the past five years are captured quite nicely by the CEV model. We visualize the real world gold data, and compare it to the stock paths generated by the CEV model. A beta of around 0.12 seems to capture the volatality of gold in recent times quite well.
